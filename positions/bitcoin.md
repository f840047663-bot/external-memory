## 宏观定位

```
BTC = 黄金的弹性版

黄金（稳定防守）←──────────────────→BTC（极限进攻）
  保命用的                             赌流动性拐点的
  跌不了多少                           能跌30-60%也能涨3-5倍
```

### 对BTC有利
1. **减半周期**（2024年已减）→ 供给减少
2. **ETF通过** → 机构资金结构性入场
3. **非负债属性** → 不需要信任任何人的偿付能力
4. **无国界** → 港股通道关闭后，少数能自由交易的资产

### 对BTC不利
1. **美债5.18%+利率高位** → 流动性紧缩→BTC被第一个卖
2. **沃什鹰派+可能加息** → 流动性拐点遥遥无期
3. **美元走强** → BTC以美元计价，承压
