# 改良PEG+PB组合估值法（2026-06-06 用户确立）

## 适用场景
适用于**大额费用化投入、当期利润人为压低、资产持续扩张**的企业估值。
典型标的：亚马逊、重投入型科技公司、前期烧钱后期爆发的平台型企业。

## 传统PE失效原因
1. 企业主动将仓储基建、物流设备、服务器算力、研发支出当期费用化 → 净利润被刻意做低 → PE分母失效
2. 投入前置、利润后置：当期花钱扩产能，收益在未来数年释放，账面净利不匹配营收增长
3. 业务分层（电商薄利+AWS前期投入），整体净利被拉低，静态PE无参考意义

## 改良PEG公式

```
改良PEG = EV / FCF ÷ FCF复合增速

标尺：
  ≈ 1 → 合理估值中枢
  < 1 → 估值偏低（成长未被充分定价）
  > 1 → 透支远期增长
```

**核心替换：** 用EBIT/自由现金流(FCF)替代账面净利润，绕开会计费用化对利润的人为调节。

**逻辑：** 资本开支转化为经营性资产（仓储、机房），不会凭空损耗，最终通过营收放量转化为持续现金流。现金流增速真实反映投入落地效果。

## PB约束（PPB思路）

改良PEG只衡量增速，无法管控无效投资 → 搭配PB做约束。

```
合理PB = f(ROE, 永续增速, 折现率)

判断：
  - 持续大额投入但ROE走低 + PB异常走高 → 资产回报率下滑 → PEG再优秀也有泡沫
  - PB回落至历史分位低位 → 即便短期FCF增速走弱，存量净资产提供下跌安全垫
```

**作用：**
1. 锚定历年累计投入形成的账面净资产
2. 排查盲目低效扩产
3. 通过ROE-PB匹配度验证资产盈利能力
4. 提供安全边际

## 与现有P值系统的关系

| 维度 | P值系统 | 改良PEG+PB |
|:----|:----|:----|
| 定位 | 信息温度计（情绪/动量） | 基本面估值（成长/资产效率） |
| 输入 | 事件链+贝叶斯 | FCF+增速+ROE+PB |
| 输出 | P值（上涨动力概率） | PEG倍数+PB合理性 |
| 适用 | 所有持仓（含ETF） | 个股/重投入型企业 |
| 互补 | 看方向 | 看价格是否合理 |

**使用场景：** 当P值显示上涨动力强（P>60%），用改良PEG+PB判断当前价格是否已经透支。PEG<1且PB合理→果断上；PEG>1.5→P值再高也要等回调。
