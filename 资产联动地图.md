# 资产联动地图（一次写好，永不过期）
> 分析任一资产前，先读此文件了解联动关系，再跑db_quick_landscape()看多空对比。

## 一、联动阵营

### 阵营1：AI/科技集群（同向联动）
```
芯片 ↔ AI ↔ 存储 ↔ 恒科 ↔ 电池（AI能源）
```
- **同向联动**：芯片涨→AI涨→恒科涨→存储涨→电池涨。反之亦然。
- **驱动力**：宏观利率压制/释放 + AI产业进展（供需/成本/人才）
- **领先指标**：美光财报、英伟达股价、韩国半导体指数 > A股芯片（滞后）
- **注意**：恒科还受南向资金独立驱动（与A股芯片有跷跷板可能）

### 阵营2：资源/商品集群（同向但逻辑有分化）
```
有色 ↔ 稀金 ↔ 原油 ↔ 能源
```
- **同向联动**：强美元压全体，弱美元拉全体
- **⚠️ 逻辑分化**：
  - 有色（铜铝锡）= 工业需求+美元定价。受利率敏感
  - 稀金（稀土锂钴）= 中国战略话语权+出口管制。稀土逻辑独立于利率
  - 原油 = OPEC+地缘+美伊协议。与有色部分联动但主导因素不同
- **黄金**：属于资源集群但逻辑特殊（见阵营3）

### 阵营3：避险/货币替代集群
```
黄金 ↔ 避险 ↔ 美元信用
```
- **黄金特例**：受两股力拉扯——长期（央行购金+美元信用担忧=利好）vs 短期（利率压制=利空）。当前处于"撕裂格局"
- **与AI集群的跷跷板**：风险偏好高→资金从黄金流向AI；风险偏好低→资金从AI流向黄金

### 阵营4：防御/独立逻辑集群
```
养殖（完全独立，猪周期自循环）
银行（A股低估值防御，受利率影响）
消费（受居民收入+社零影响）
券商（受益于A股交易量）
```

### 阵营5：宏观/外部驱动力（影响所有资产）
```
利率/美联储 → 影响所有风险资产（最核心）
美元 → 压制大宗商品定价
地缘 → 支撑黄金/原油避险溢价
美股/纳指 → 通过情绪传导影响恒科/AI芯片
```

## 二、跨阵营传导关系

| 源头 | 传导到 | 逻辑 | 时效 |
|:----|:-------|:----|:----|
| **利率↑** | 芯片/AI/恒科↓ 有色↓ 黄金(短期)↓ 养殖↔(无关) | 强美元压制风险偏好 | 即时 |
| **AI芯片↑** | 恒科↑ 电池↑ 有色(铜需求)↑ 存储↑ | AI基建拉动全产业链 | 1-3个月 |
| **AI芯片↓** | 恒科↓ 电池↓ 但→有色/稀金部分独立(需求逻辑不变) | AI回调→情绪传导 | 即时 |
| **避险↑** | 黄金↑ 美元↑ 芯片/AI↓ 恒科↓ | risk-off | 即时 |
| **养殖(猪周期独立)** | 其他一切↔ | 猪周期=独立供需，不受利率/避险/AI任何影响 | 永远独立 |
| **稀土逻辑(出口管制)** | 稀金↑ 但→有色↔(不联动) | 政策驱动，独立于商品周期 | 数月 |

## 三、关键铁律

1. **养殖是孤岛**：任何时候分析养殖都不需要考虑其他资产。猪周期=独立供需闭环。
2. **稀金≠有色**：稀土战略逻辑走政策，不走利率。分析稀金时用有色框架会错。
3. **黄金撕裂**：长期逻辑（央行购金+美元信用）vs 短期（利率压制）。不同时间维度的结论可能相反。
4. **AI/芯片现在最大的风险不是估值，是"供给→需求拐点"（付鹏6/26）**：上游涨价把下游（苹果）吃死了。这不是回调，是结构性问题。
