# 日报作业指导书

## 触发条件
监控流程结束后产出日报

## 前置检查
- 必须先检查events文件里独立事件数+看板新增行数，≥5条才出日报
- 低于5条保持安静

## 执行步骤
1. [ ] 步骤一：读转录内容
2. [ ] 步骤二：精炼写看板+记临时笔记
3. [ ] 步骤三：通读笔记
4. [ ] 步骤四：串主线
5. [ ] 步骤五：出日报正文（三段式：主线摘要→各赛道梳理→明日关注）
6. [ ] 步骤六：投递（邮件或微信）

## 检查点
- 事件数和看板新增行数是否≥5
- 笔记是否已完整整理
- 主线是否清晰可串
- 日报正文三段式结构是否完整

## 完成标准
日报已投递到用户
