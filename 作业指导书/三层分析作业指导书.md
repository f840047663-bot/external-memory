# 三层分析作业指导书

## 触发条件
用户要求研究某资产/调出某资产最近情况/开始跟踪新资产

## 前置检查
确认是哪个资产（从asset对照表里选）

## 执行步骤
1. [ ] 步骤一：宏观层（利率/地缘/经济环境）
2. [ ] 步骤二：中观层（产业/能源/板块逻辑）
3. [ ] 步骤三：微观层（该资产具体事件链/P值变化）
4. [ ] 步骤四：三关检查（懂不懂/护城河/管理层）
5. [ ] 步骤五：3选1结论（不碰/定投/右侧等业绩）

**达利奥四象限检查：** 增长↑通胀↑ / 增长↑通胀↓ / 增长↓通胀↑ / 增长↓通胀↓，检查持仓是否过度集中

## 检查点
- 三层分析是否完整（宏观→中观→微观）
- 三关检查是否通过
- 达利奥四象限是否已核对
- 结论是否明确为3选1

## 完成标准
用户获得该资产的完整三层分析+操作结论
