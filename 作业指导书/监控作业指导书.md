# 监控作业指导书

## 触发条件
每天13:00 cron触发/用户说"跑监控"/用户说"加载master-routing"

## 前置检查
- 五个开关检查（抖音/B站/知乎/财联社/其他）
- chrome是否可达
- cdp-bridge是否正常

## 执行步骤
1. [ ] 步骤一：抓取（抖音/B站/知乎/财联社）
2. [ ] 步骤二：转录（asr转文字）
3. [ ] 步骤三：分析（提炼关键信息）
4. [ ] 步骤四：归档（调用归档模块）
5. [ ] 步骤五：更新看板

**sheet2列对应：** 1芯片 3恒科 4养殖 5有色 6稀金 7银行 8AI
**总看板列对应：** 4经济 5能源 6产业 7避险

## 检查点
- 每步完成后检查日志，失败自动重试1次
- **断点铁律：** 看板没更新=没做完。修改Excel前必须先cp备份（带时间戳）

## 完成标准
看板Excel已更新+数据库已写入
