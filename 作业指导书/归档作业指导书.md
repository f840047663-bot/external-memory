# 归档作业指导书

## 触发条件
收到任何投资相关事件/信号/视频

## 前置检查
- asset对照表：AI, 其他, 养殖, 原油, 存储, 恒科, 有色, 核电, 电池, 稀金, 芯片, 银行
- sheet分类：Sheet2（赛道持仓跟踪）, 总看板（宏观分类）
- 总看板分类：利率, 地缘, 经济, 能源, 产业, 避险

## 执行步骤
1. [ ] 步骤一：从用户发来的内容中提取独立事件，每个事件一个atom
2. [ ] 步骤二：每个事件写入events表（asset/category/sheet/date/p_value/content/comment/source_ver）
3. [ ] 步骤三：同时写入事件文件（按asset分文件，存放在~/investment_workdir/events/目录）
4. [ ] 步骤四：更新看板Excel（Sheet2和总看板都更新）
5. [ ] 步骤五：写看板前必须cp备份带时间戳

## 检查点
- 每个事件是否已正确提取为独立atom
- 数据库写入字段是否完整（asset/category/sheet/date/p_value/content/comment/source_ver）
- 事件文件是否按asset正确分类存放
- 看板更新前是否已备份（带时间戳）

## 完成标准
数据库events表有记录 + 事件文件存在 + 看板Excel已更新（三项缺一不可）
