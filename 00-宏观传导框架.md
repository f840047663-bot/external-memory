## 一、利率传导路径

### 核心机制

全球利率中枢受美联储政策 + 各国债务结构 + 通胀预期三重影响。

```
美联储加息/维持高位
  ↓
全球无风险利率上升
  ├→ 债券价格下跌（存量债折价）
  ├→ 股票估值承压（折现率上升→DCF模型估值下降）
  ├→ 企业融资成本上升（利空高杠杆行业）
  └→ 银行净息差改善（利好银行息差收入，但坏账风险同步上升）
```

### 当前状态（2026-05-23 ※沃什就任+系统性折价系数已录入）

- **🔴 系统性折价系数：-4pp（10年美债收益率约4.5%-5.0%区间，偏紧环境）**
- **🔴 30年美债拍卖中标收益率5.046%，2007年以来首次破5%（宋鸿兵5/18转录确认）** — 跟盘中交易收益率不同，拍卖中标收益率破5%是第一次。周五30年美债收益率飙升到5.12%。10年美债突破4.5%红线，盘中接近6%。
- **环中星鉴（2026-05-20）补充论据：** 美国财政部年赤字2万亿（占GDP6%），利息支出超1万亿吃掉25%财政收入。2026年9-10万亿疫情期低息国债到期，需在高利率环境再融资→财政部根本撑不住持续加息。这是比"利率压制估值"更深层的宏观逻辑——不是市场愿意加息，是美国政府加不起。
- **韩国股市暴跌6%**，三星-8.6%、SK海力士-7.7%（这两只贡献今年韩国全部涨幅的2/3）。AI牛市在利率风暴冲击下显露脆弱。
- 日本30年国债历史上首次突破4%，达4.08%。英国3年国债5.85%创30年最高。德国30年国债3.7%创2011年来最高。
- **根本原因：** 伊朗战争→霍尔木兹海峡封锁→能源价格飙升→通胀预期上升→市场预计12月底前美联储加息概率接近40%（沃什上任面临的问题不是降息，而是是否重新加息）
- **宋鸿兵核心判断：** 全球最安全的资产（欧美日国债）在经历系统性重新估价，其他风险资产无法独善其身。

## 二、地缘政治传导路径

### 核心机制

```
地缘冲突（中美/俄乌/伊朗/台海）
  ↓
不确定性上升
  ├→ 避险资金流入黄金/美元
  ├→ 能源价格波动（尤其原油/天然气）
  ├→ 供应链中断风险（芯片/稀土/关键矿产）
  └→ 风险资产整体承压（股市下跌）
```

### 当前状态（2026-05-19）

- 伊朗局势持续紧张，原油供给端承压
- 中美关系：习近平访美（缓和信号），但结构性矛盾未变
- 全球地缘不确定性指数处于历史高位
- **⚠️ 闻号说经济（2026-05）提出反共识（⭐⭐）：** 华尔街通过大量发行贵金属ETF/期货/期权已将黄金从\"价值储存器\"变成\"风险放大器\"。全球黄金ETF持仓超官方储备总和，黄金与纳指相关性增强。**如果此论点为真→黄金对冲属性被结构性削弱→宏观传导路径中的黄金利好逻辑需要下调。** 与环中星鉴/宋鸿兵/付鹏的多头框架对立，设为观测点。

## 三、能源价格→成本传导路径

### 核心机制

```
霍尔木兹海峡封锁 / 地缘冲突导致油价飙升
  ↓
原油价格上涨
  ├→ 航运燃料成本↑ → 国际海运运费↑（巴西/美豆→中国运距长，影响最大）
  │       ↓
  │   大豆/玉米进口成本↑ → 饲料成本↑ → 养殖利润被压缩（利空养殖ETF）
  │
  ├→ 化肥原料（天然气）成本↑ → 全球化肥价格↑ → 农产品种植成本↑（远期）
  │       ↓
  │   闻号说经济：化肥暴涨60%（2026-05-18），传导到饲料端需2-3个季度
  │
  ├→ 化工原料成本↑ → 电池材料/隔膜等成本↑（短期影响小）
  │
  ├→ 通胀预期↑ → 央行被迫加息/不降息 → 利率环境收紧
  │       ↓
  │   见「一、利率传导路径」
  │
  └→ 产油国财政改善 → 地缘博弈空间扩大（霍尔木兹封锁更持久）
```

### 当前状态（2026-05-23）

- **霍尔木兹海峡：部分封锁中** — 伊朗战争导致航运受限
- **WTI原油** ~96.5美元/桶，本周累跌8.4%（有缓和预期）
- **沙特媒体称获美伊协议草案** — 停火+航行自由（2026-05-22）
- **化肥暴涨60%**（闻号说经济 05-18）— 但化肥到饲料有2-3季度滞后
- **运价指数未明显跳升** — 当前航运成本传导尚未到临界点

## 四、经济周期传导路径 + 结构性转型（瑞·达利欧 新旧二元论）

### 核心机制

```
经济走弱（社融负值/消费疲弱/PMI下行）
  ↓
企业盈利预期下降
  ├→ 银行：贷款需求萎缩 + 坏账率上升（利空）
  ├→ 消费类：内需不足（利空养殖/消费）
  ├→ 政策刺激预期上升（利好债券/基建）
  └→ 避险情绪上升（利好黄金）
```

### 结构性转型：新旧二元论（瑞·达利欧框架 2026-05）

> 达利欧认为中国正处于经济结构大转型：**旧经济（地产/高负债国企/过剩产能）必须重组，新经济（AI/机器人/新能源/高端制造/生物科技/创新药）才是未来引擎。**

```
新旧二元论（结构性框架）
  ↓
旧经济（地产链/高负债国企/过剩产能）
  ├→ 风险源：必须重组，拖累整体
  └→ 但不会硬着陆（政策管控+温和去杠杆）
  ↓
新经济（AI/机器人/新能源/高端制造/创新药）
  ├→ 增长引擎：政策倾斜+资本聚集+人才溢出
  └→ 短期被旧经济拖累，长期是唯一方向
  ↓
政策方向：温和去杠杆 + 储蓄导流新经济 + 支持民营 + 继续开放
```

**⭐ 贝叶斯评估（达利欧·最高置信度源）：**

达利欧是几十年顶级宏观大师，**置信度级别高于所有现有信源**（高于⭐⭐⭐⭐⭐级别的宋鸿兵/付鹏）。

## 五、⚠️ 传导路径限定规则（2026-05-23 新增）

> **原则：不是所有宏观因子都传导到所有资产。中国居民消费因子是杀伤面积最小的因子之一。**

### 5.1 判断框架

| 因子 | 谁能打到 | 谁打不到 |
|:----|:---------|:---------|
| 中国居民消费 | 内需消费股（养殖·有限）、可选消费 | ❌ 全球定价工业品（有色/稀有金属） |
| | 银行·部分（零售贷款） | ❌ 全球避险资产（黄金/美债） |
| | | ❌ 中国科技（有限，分成分看） |
| 全球利率 | ✅ 所有成长股、债券、黄金、银行 | 养殖（无关） |
| 地缘风险 | ✅ 黄金、原油、航运 | 消费股（无关） |
| 美元指数 | ✅ 黄金、港股、新兴市场 | 国内纯内需（无关） |

### 5.2 核心禁止规则

**禁止将中国居民消费因子应用于：**
1. **全球定价的工业商品**（有色金属、稀有金属、原油等）
2. **全球定价的避险资产**（黄金）
3. **非消费属性资产**（芯片、先进制造等，除非有明确消费端逻辑）

### 5.3 判定标准

调整P值前问三个问题：
1. 这个资产的**买家是谁**？→ 老百姓 → 消费因子有效 → 否则禁止
2. 这个资产的**定价权在哪**？→ 国内 → 消费因子有效 → 否则禁止
3. 这个资产的**需求弹性**多大？→ 必需（低弹性）→ 消费因子杀伤力有限

> 违反本规则的P值调整，月度总检时回滚并标记为"系统性偏差"。

## 六、越靠近GPU越值钱——产业链价值排序原则

> **核心原则：** 在一个AI产业链中，价值沿着"离GPU的物理/技术距离"递减。越靠近GPU的环节，议价能力越强、技术壁垒越高、资本开支门槛越大、供给越稀缺。

### 原理推演

蒋宇飞提出的 **"存储比光模块值钱的三大原因"** 是这一原则的完美案例：

```
GPU（AI原点）
  │
  ├── 存储（HBM/DRAM/NAND）← 离GPU最近
  │     ├── 全球仅7个玩家，HBM仅3家（集中度极高）← ⭐ 原因一
  │     ├── 资本开支100亿美金起（物理稀缺性高）← ⭐ 原因二
  │     ├── 技术替代难度极大（HBM需要TSV/微凸块等先进封装）← ⭐ 原因三
  │     └── 全球4000亿美金规模（光模块10倍）
  │
  ├── 堆叠（先进封装/CoWoS）
  │     ├── 玩家有限（台积电/三星/Intel）
  │     ├── 本质在替代摩尔定律
  │     └── 千亿美金级市场
  │
  ├── 光模块
  │     ├── 400家厂商（竞争激烈，无核心技术门槛）← 集中度低
  │     ├── 资本开支10-20亿够用 ← 门槛低
  │     └── 全球400亿美金，利润兑现但价格往下
  │
  └── 应用层（最远）
        ├── 进入门槛最低
        ├── 竞争最激烈
        └── 黄仁勋说的"AI真正机会在应用层"→说明基础设施已够用→应用层即将内卷
```

### 核心推论：投资重心排序

根据"越靠近GPU越值钱"原则，AI产业链的投资价值排序为：

| 排序 | 环节 | 价值依据 | 代表持仓 |
|:---:|:----|:---------|:--------|
| 🥇 | **GPU/CPU（芯）** | AI原点，不可替代，英伟达/AMD双寡头 | 海外科技007476 |
| 🥈 | **存储（存）** | 离物理距离最近+集中度最高+门槛最高 | 芯片ETF 020628 |
| 🥉 | **堆叠（叠）** | 替代摩尔定律，千亿美金市场 | 芯片ETF 020628 |
| 4 | **光模块（光）** | 规模小且无门槛，利润见顶 | 海外科技007476（低权重） |
| 5 | **应用层** | 门槛最低，黄仁勋点名的方向但内卷在即 | 无直接持仓 |

### 对当前持仓的指导意义

1. **芯片ETF 020628** 🟢 核心受益 — 芯+存+叠三条高价值线，按原则是AI投资的最优大类
2. **海外科技007476** 🟡 中期中性 — 芯(GPU)在海外，∴价值最高，但光模块部分价值递减；且海外估值溢价被利率压制

4. **如果未来要减AI仓位** → 优先减海外科技（光模块价值层偏低，且海外利率敏感度更高）

## 七、AI负反馈循环：生产力-生产关系错配（大好人的1929类比框架）

> **建档：** 2026-05-24 | 大好人提出，基于当前AI发展与1929大萧条的结构性相似性
> **核心矛盾：** 生产力借助AI快速跃升，但生产关系（就业/分配/消费）未同步适配，短期内不仅不带动经济，反而引发负反馈循环。

### 核心机制

```
AI生产力提升
  ├→ AI替代人力 → 就业缩减 → 居民收入下降 → 消费持续萎缩（🔴 负反馈启动）
  ├→ 企业被迫AI军备竞赛 → 巨额算力/技术投入 → 运营成本居高不下
  │       ↓
  │  消费疲软 → 企业收益承压 → 进一步裁员降本 → 消费更弱（🔄 恶性循环加剧）
  │
  ├→ 行业内卷 → 不跟进AI = 被淘汰 → 被迫加码投入 → 成本持续上升
  │
  └→ 本质：生产力跃升了，生产关系（分配/就业/消费结构）没跟上
```

### 1929结构性相似点对照

| 维度 | 1929年 | 2026年AI时代 |
|:----|:-------|:------------|
| 产能 vs 消费 | 工业产能过剩，民众无力消费 | AI替代人力→失业→消费萎缩 |
| 技术冲击 | 流水线提升效率→裁员→消费下降 | AI替代白领/蓝领→同等效果 |
| 企业行为 | 被迫降价/减产，但成本刚性 | 被迫AI投入，但收益承压 |
| 政策困境 | 政府缺乏现代宏观调控工具 | 当前政策空间有限（高债务+高利率） |

### 🔑 关键区分：破坏性创造 vs 破坏性优化

| | 破坏性创造（Creative Destruction） | 破坏性优化（Destructive Optimization） |
|:----|:----------------------------------|:--------------------------------------|
| 定义 | AI消灭旧岗位，但创造新产业/新岗位 | AI消灭岗位后，没有创造对等的新价值链 |
| 结果 | 短期阵痛→长期新增长 | 只有阵痛，没有后续增长 |
| 历史案例 | 工业革命→工厂工人→服务业兴起 | 日本"失落的30年"（资产负债表衰退） |
| ⚠️ **当前观察** | ❌ 未看到 | ✅ 只看到破坏性——企业裁员降本、消费萎缩、AI替代而不创造新需求 |

**目前只看见了破坏性，没看见创造性。** 如果出现破坏性创造（AI催生出新产业/新就业形态），负反馈循环会被打破。但当前信号全指向破坏性优化方向。

### 负反馈循环对持仓的传导

```
AI负反馈循环
  │
  ├→ 消费萎缩
  │     ├→ 利空：养殖ETF（内需承压）
  │     ├→ 利空：银行（零售贷款需求弱）
  │     └→ 中性：电池（电动车可选消费承压，但储能逻辑独立）
  │
  ├→ AI成本高企 → 企业盈利承压
  │     ├→ 利空：芯片ETF（博主的AI乐观与宏观负反馈的对立——需降权处理）
  │     ├→ 利空：海外科技、纳指（估值中的AI溢价部分可能被宏观证伪）
  │     └→ 利空：永赢/平安先进制造（机器人长期逻辑成立，但短期需求端被消费萎缩压制）
  │
  ├→ 利率/政策空间受限
  │     ├→ 利空：所有风险资产（政策无法有效刺激）
  │     └→ 利好：黄金（避险+法币信用动摇）
  │
  └→ 风险偏好下降
        └→ 利空：恒生科技（风险偏好最敏感）
```

### 与达利欧新旧二元论的关系

> **达利欧说：** 旧经济必须重组，新经济（AI/机器人/新能源）是唯一方向。
> **大好人说：** AI新经济本身在创造负反馈循环——生产力跑太快，生产关系没跟上。
>
> 两者不矛盾，是同一枚硬币的两面：
> - **达利欧看到的是方向（长期）** — 新经济取代旧经济是必然
> - **大好人看到的是路径中的陷阱（短期-中期）** — 在AI替代旧岗位→新岗位被创造出来之前，有一段"破坏性优化"期
>
> **结论：** AI方向长期正确✅，但短期-中期的负反馈循环可能造成比预期更深的调整 ⚠️

### ⚠️ 贝叶斯影响

本框架是**宏观背景假设**，不是单一事件。对P值的影响：
- **不触发单次pp调整**（这不是一条"新信息"，是系统级背景）
- **作为系统性折价/溢价的参考** — 在评估AI相关资产（芯片/海外科技/纳指/永赢/平安）时，需在5道门的B门(宏观折价)中考虑负反馈循环的打折系数
