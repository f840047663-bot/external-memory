# 盈亏比计算方法论

> **最后更新：** 2026-06-03
> **来源：** 用户定稿
> **关联：** 00-估值方法论.md（丛林三问）+ 看板P值系统 + 前瞻扫描

---

## 核心定义

**盈亏比 ≠ P值。** 两者拆分两层定量算：

| 概念 | 回答什么问题 | 用什么算 |
|:----|:----------|:--------|
| **P值** | 方向对不对？（信息面多空概率） | 事件链 → 贝叶斯更新 |
| **盈亏比** | 值不值得干？（上行空间 vs 下行空间） | 估值锚定（PE/PB），剔除看图主观 |

**决策铁律：P值 ≥ 60% + 盈亏比 ≥ 2:1，双条件满足才下单。**

---

## 一、下行空间（止损锚：下跌极限）

> 用历史估值下沿做客观止损，不靠"看图"。

**公式：**
```
下行空间 = 现价 - 历史估值下沿
```

**历史估值下沿 = 两层底线取低者：**
1. **PE/PB 历史 10% 分位数**（近 5-10 年数据）
2. **净资产底线**（PB ≤ 1 或接近 1）

**极端情况再加一层：**
3. 行业景气下修带来的估值杀跌预留（如周期股景气谷底）

**用途：** 这是客观最大亏损幅度，用来做风控止损位。

---

## 二、上行空间（分两段）

### 2.1 确定性空间（合理估值上沿）

> 业绩兑现就能摸到的保守目标，实打实的基本面收益。

```
确定性空间 = 合理估值上沿 - 现价
合理估值上沿 = 近5年中枢PE/PB × 当期业绩增速匹配
```

### 2.2 溢价空间（想象空间）

> 戴维斯双击的超额溢价，属于弹性收益。

**来源：**
- 行业政策利好（如半导体国产替代加速）
- 产能扩张周期启动
- 新产品/新技术突破
- 赛道景气度提升（估值从中枢打到历史高位分位）

```
溢价空间 = 历史高位分位估值 - 合理估值上沿
```

### 2.3 P值与溢价空间的联动（事件驱动法整合点）

> **这是事件驱动法和你盈亏比框架的吻合处。**
> P值追踪的事件链（政策/景气/新产品）= 溢价空间的来源。
> P值越高，说明事件链越支持上行，溢价空间越能大胆算。

| P值区间 | 溢价空间打折系数 | 逻辑 |
|:-------|:-------------|:----|
| P ≥ 70% | 100%（不打折） | 事件链强支持，溢价大概率能吃到 |
| P 60-70% | 50% | 事件链中等支持，溢价打五折 |
| P < 60% | 0%（不算溢价） | 事件链不支持，只看确定性空间 |

---

## 三、空间盈亏比公式

```
盈亏比 = (确定性空间 + 想象溢价空间 × P值打折系数) ÷ 下行极限空间
```

**决策标准：**
- 盈亏比 ≥ 2:1 → 值得干
- 盈亏比 1.5-2:1 → 勉强，需要P值很高（≥75%）才考虑
- 盈亏比 < 1.5:1 → 不干

---

## 四、关键取舍

### 保守交易
- 只算合理估值一段（确定性空间），剔除想象空间
- 盈亏比稳健、容错高
- 适合：左侧布局、不确定性强时

### 波段博弈
- 叠加景气想象溢价，上行空间放大
- 但要下调止损空间（更严格的止损），严控盈亏比底线 ≥ 2:1
- 适合：右侧确认、P值高时

---

## 五、不同资产类型的估值锚

| 资产类型 | 估值指标 | 约束条件 | 说明 |
|:--------|:--------|:--------|:----|
| **周期股**（有色/养殖/原油） | PB + 产能周期 | 盈利周期性失真，PE不可靠 | 用PB历史分位 + 产能周期位置测算上下沿 |
| **成长股**（芯片/AI/电池） | PE | PEG约束估值上限 | 防止想象空间虚高，PEG > 1.5 打折 |
| **金融股**（银行/保险） | PB | 股息率做底线 | PB ≤ 0.8 + 股息率 ≥ 5% = 安全区 |
| **宽基/行业ETF** | PE/PB 历史分位 | 指数成分决定 | 直接看指数PE/PB历史百分位 |

## 六、与看板P值系统的整合

### 完整决策流程

```
Step 1: P值判断方向
  → 看板事件链 → 贝叶斯更新 → P值 ≥ 60%？
  → 否 → 不操作
  → 是 → 进入Step 2

Step 2: 盈亏比判断值不值
  → 算下行空间（估值下沿）
  → 算上行空间（确定性 + 溢价，溢价按P值打折）
  → 盈亏比 ≥ 2:1？
  → 否 → 不操作
  → 是 → 进入Step 3

Step 3: 事件驱动定止损/止盈
  → 止损 = 下行空间对应的价格（估值下沿价）
  → 止盈1 = 确定性空间对应的价格（合理估值价）
  → 止盈2 = 溢价空间对应的价格（历史高位价）
  → 右侧信号确认 → 止损上移到信号位
```

## 七、与丛林三问的对应

| 丛林三问 | 盈亏比框架 | 说明 |
|:--------|:----------|:----|
| 有没有鸟？ | 下行空间有底线吗？ | 估值下沿 = 鸟还在不在丛林里 |
| 什么时候飞？ | 催化剂/事件链 | P值追踪的事件链 = 风什么时候来 |
| 有多少只？ | 确定性空间 + 溢价空间 | 上行空间 = 有多少只鸟能飞出来 |
| 无风险利率 | 盈亏比 ≥ 2:1 | 不划算就是最大的风险 |

## 八、实操示例模板

> 以下为模板，实际数据由AI抓取填充。

### 示例：芯片ETF 020628（假设数据）

```
当前数据：
- 现价：1.20
- 近5年PE 10%分位：20x → 对应价格 0.80
- 近5年PE 中枢：35x → 对应价格 1.46
- 近5年PE 90%分位：50x → 对应价格 2.08
- 当期业绩增速：15%
- P值：65%

计算：
- 下行空间 = (1.20 - 0.80) / 1.20 = 33%
- 确定性空间 = (1.46 - 1.20) / 1.20 = 22%
- 溢价空间 = (2.08 - 1.46) / 1.20 = 52%
- P值65% → 溢价打五折 = 26%
- 盈亏比 = (22% + 26%) / 33% = 1.45:1

结论：
- 保守模式（不算溢价）：22%/33% = 0.67:1 → 不干
- 波段模式（溢价五折）：48%/33% = 1.45:1 → 不干
- 需要P值更高或下行空间更小才值得
```
