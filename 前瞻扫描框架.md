# 前瞻扫描制度化框架

## 为什么需要这个
大好人右侧交易，不亏本为第一原则。这意味着：
- 不能在涨了才追 → 要提前知道什么在低估区
- 不能在跌了才跑 → 要提前感知风险信号
- 现有系统全是持仓管理（跌了怎么办），没有"外面还有什么"的扫描机制

## 扫描三支柱

### 1️⃣ 流动性温度计 — 比特币
**频率：** 每次前瞻扫描必看
**数据源：** 财联社搜"比特币"+ 价格走势
**信号判断：**
| BTC走势 | 含义 | 对持仓影响 |
|:--------|:----|:-----------|
| 连续上涨 | 流动性宽松，风险偏好高 | 🟢 利好所有风险资产 |
| 连续下跌 | 流动性收紧，避险 | 🔴 小心海外科技/芯片 |
| 横盘震荡 | 方向不明 | 🟡 观望 |
| 比特币跌破前低 | 流动性极度紧缩 | ⚠️ 考虑减仓 |

### 2️⃣ 黄金温度计 — 全球资产占比+实际利率
**频率：** 每周一次
**数据源：** 财联社搜"黄金"+"金价"+"央行购金"
**温度计：** 黄金在全球金融资产中的占比（正常区间约0.5-1%）
| 信号 | 判断 | 含义 |
|:----|:----|:-----|
| 占比低 + 实际利率降 | 🟢 利好 | 金价有上行空间 |
| 占比高 + 实际利率升 | 🔴 利空 | 金价压力大 |
| 央行持续购金 | 🟢 结构性利好 | 长期需求支撑 |

### 3️⃣ 板块估值雷达
**频率：** 每周一次
**数据源：** 东方财富ETF净值 + 财联社行业新闻
**覆盖板块（初期）：**
- 银行 ETF（已有监控）
- 芯片 ETF 020628
- 有色金属 008826
- 新能源/绿色电力
- 医药
- 消费
- 恒生科技
- **以后发现新板块再加**

**判断标准：**
**核心原则：** 历史百分位 = 温度计（参考），**逻辑 = 最终判断依据**

**不同资产用不同指标：**
| 资产类型 | 温度计指标 | 风险提示 |
|:--------|:---------|:--------|
| 银行/金融 | PB历史百分位 | PB<1=折价，全球通用 |
| 成熟制造业 | PE历史百分位 | 业绩稳定有效 |
| 科技/芯片/海外 | **PEG历史百分位** | ⚠️ 必须检查泡沫风险，PEG低也可能是因为增长没了 |
| 周期股（有色/养殖） | PB（底部时） | 亏损时PE无效 |
| 恒生科技 | 指数PE百分位 + PEG | 成分股复杂，两者结合 |
| 黄金 | 不适用 | 看全球资产占比+实际利率 |

**逻辑 vs 估值优先级：**
| 逻辑状态 | 百分位 | 判断 | 动作 |
|:--------|:------|:----|:----|
| ✅ 逻辑没病 | 🔴 低位区 | 低估+有道理 | ✅ 可以买 |
| ✅ 逻辑没病 | 🟡 中位区 | 合理估值 | 🟡 有触发条件再考虑 |
| ✅ 逻辑没病 | 🟢 高位区 | 好东西但贵了 | 🟡 等回落到中位区 |
| ❌ 逻辑变了 | 🔴 低位区 | ⚠️ **价值陷阱** | ❌ **绝对不能碰** |
| ❌ 逻辑变了 | 任何位置 | 逻辑已死 | ❌ 卖掉 |

### 4️⃣ 宏观先行指标
**频率：** 每周一次
**数据源：** 财联社搜索
**扫描清单：**
| 指标 | 看什么 | 为什么重要 |
|:----|:------|:----------|
| MLF/ LPR利率 | 降息了没 | 流动性宽松=利好 |
| PMI | 连续3个月走势 | 经济在复苏还是收缩 |
| 社融数据 | 超预期还是低于预期 | 企业愿意借钱=经济在动 |
| 美债收益率 | 升还是降 | 影响海外科技持仓 |
| 人民币汇率 | 贬值还是升值 | 影响QDII/港股 |
| 实际利率 | TIPS收益率/国债利率-通胀 | 黄金的机会成本，负利率=黄金利好 |
