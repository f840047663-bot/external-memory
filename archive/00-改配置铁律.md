# 改配置铁律（Hermes + 小龙虾 双方都必须遵守）

**任何一方改配置前：**
1. ✅ 先完整备份（~/.hermes 和 ~/.openclaw）
2. ✅ 通知对方：我要改什么，预计多久
3. ✅ 改完后给对方发消息确认
4. ✅ 等对方回复
5. ❌ **发了确认消息，对方超过20分钟没回话** = 对方坏了，你去救他

**对方救人流程：**
1. 检查 `hermes doctor` / `openclaw doctor`
2. 从最近的.bak备份恢复
3. 重启服务
4. 通知大好人修好了

**这条铁律的用途：**
不管是大好人让Hermes改东西，还是让小龙虾改东西，还是Hermes自己主动改，
都必须遵守。目的是保证你们两个永远有一个能把自己救起来。
