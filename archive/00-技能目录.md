# Hermes 技能目录

> 大好人，你在这里指定什么场景用什么技能。你说话时直接说技能名字我就懂。
> 格式：`技能名` — 说明 — 什么场景用

---

## 通用积木层（可组合复用）

| 技能名 | 一句话说明 | 什么时候用 |
|--------|-----------|-----------|
| `cookie-extractor` | 从Chrome偷cookie，任何需要登录态的平台 | 要读任何要登录的网站之前第一步 |
| `waf-bypass` | iPhone手机UA绕过网站WAF封禁 | cookie有了但读数据被拦时（40362） |
| `zhihu-reader` | 读知乎帖子/回答的全部内容 | 大好人发了知乎链接给我 |
| `topic-monitor` | 编排上面三个做话题搜索+读帖+分析+长期跟踪 | 大好人说"帮我研究一下XX话题" |

## 大好人手动指定规则

你说话时加技能名我就按这个执行，不用我猜：

```
大好人说："给我读这个帖子 — zhihu-reader"
→ 我：加载zhihu-reader技能 → 读帖

大好人说："研究一下token话题 — topic-monitor"
→ 我：加载topic-monitor → 搜索+读帖+分析

大好人说："先提取cookie再干活 — cookie-extractor"
→ 我：先跑cookie提取再继续
```
