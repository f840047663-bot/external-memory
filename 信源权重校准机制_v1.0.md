# 信源权重校准机制 v1.0
# 最后更新：2026-06-05 23:45

## 铁律1：不准只看价格结果就判信源对错

**错误示范（2026-06-05踩坑）：**
伊娃说仔猪420元+猪企亏损→供给收缩，养殖ETF跌了15%，直接判她"2/2全错"。

**正确流程：**
```
第1步：事实核查（独立于价格）
  - 查中间环节数据：仔猪价格/财报/产能数据是否准确？
  - 事实准→事实准确率=高；事实歪→事实准确率=低；查不到→暂不调整

第2步：传导分析（事实→价格的桥梁）
  - 即使事实准，价格也可能不反应：
    a) 传导滞后：产业事件需30-90天，宏观事件5-15天，情绪事件1-3天
    b) 宏观覆盖：利率/关税压制估值，产业利好被覆盖
    c) 需求恶化：供给收缩但消费更差
  - 传导预测力 = 事实最终对价格的影响程度

第3步：权重 = f(事实准确率, 传导预测力)
  - 事实准+传导强 → 高权重
  - 事实准+传导弱 → 中权重（事实有价值但价格反应慢）
  - 事实歪 → 低权重
  - 事实未知 → 暂不调整
```

## 铁律2：信源权重是二维矩阵

信源×持仓分别算权重。付鹏对芯片的权重≠付鹏对养殖的权重。

## 铁律3：Spearman秩相关做权重校准

```
X = 信源事件PP序列
Y = 对应持仓在传导窗口L天后的价格涨跌序列
ρ = spearmanr(X, Y) → 权重
```

传导窗口L按事件类型默认：
- 产业事件（财报/产能）：L=30-90天
- 宏观事件（利率/关税）：L=5-15天
- 情绪事件（闪崩）：L=1-3天
- 样本≥3时用滞后相关扫描[0,30]天自动估计

样本<2：权重=1.0；2-4条：算但标注低置信度；≥5条：正式生效。

## 铁律4：虚P不剔除

虚P（预期事件）参与P值计算（索罗斯反身性：预期推动现实）。但标记"虚"，不参与事实准确率评估。虚P长期不被价格验证→该信源传导预测力衰减。

## 铁律5：任何"拟合不好"必须深度分析原因

不准说"拟合0分→P值不可信"就完了。必须分析：
1. 事件链漏了哪些关键事件？
2. 哪些PP强度定得不合理？
3. 哪些信源事实层没核查？
4. 传导窗口是否估计错误？
5. 宏观干扰是否覆盖了产业信号？

## 铁律6：权重校准是持续过程

每次新事件→深度分析→分配PP→查权重→算P值
每次新价格→验证未验证事件→重算Spearman→更新权重
