# 决策日志索引

## 用途
记录大好人每一次投资决策（买/卖/加仓/减持/观望），用于每两周一次的「决策回看」——对比决策时的逻辑和后续结果，给信息源做贝叶斯打分。

## 结构
```
decisions/
├── INDEX.md              ← 本文件
├── 2026-05-21.md         ← 每次决策记一条
└── ...
```

## 每条决策格式
```
## 2026-05-21 14:30
**类型：** 买入/卖出/加仓/减持/观望
**资产：** 芯片ETF 020628
**金额：** 无（仅观望）
**逻辑：** 当时做这个决策的理由
**P值（当时）：** 58%
**信息源触发：** 蒋宇飞（芯光存叠主线）
```

## 回看流程
1. 每两周 cron 触发 → 读取过去14天所有决策
2. 遍历每条决策 → 对比当前结果（盈亏/逻辑验证）
3. 给每个信息源打分（准了加分/错了减分）
4. 输出回看清单 → 存入 events/ 文件
5. 将信源评分更新到对应的 thinker 文件
