# 阶段一抓取（三平台交替拉API）

## 输入
| 参数 | 类型 | 说明 |
|:----|:----|:----|
| 调试模式 | 要/不要 | 来自第〇步开关甲 |

## 输出（返回值）
```
{
  "状态清单": "/tmp/monitor_status.md",
  "统计": {"成功": X, "失败待重试": Y, "0新": Z},
  "待转录列表": ["视频ID列表"],
  "待分析列表": ["知乎文本路径列表"]
}
```

## 执行步骤

### Step 1：初始化暂存目录
```bash
mkdir -p ~/.hermes/work/videos      # 待转录的视频
mkdir -p ~/.hermes/work/texts       # 知乎文章/回答
mkdir -p ~/.hermes/work/transcripts # 转录完成的文字

# 初始化抓取状态清单
echo "# 抓取状态清单 $(date +%Y-%m-%d_%H:%M)" > /tmp/monitor_status.md
echo "" >> /tmp/monitor_status.md
echo "| 平台 | 博主 | 类型 | 结果 | 新内容数 | 暂存文件 | 备注 |" >> /tmp/monitor_status.md
echo "|:----|:----|:----|:----|:--------:|:--------|:----|" >> /tmp/monitor_status.md
```

### Step 2：读断点
打开`/home/fw/监控处理断点.md`，拿每个博主最后视频ID+时间。

**⚠️ 断点重建铁律（文件损坏/丢失时）：**
```
✅ 正确数据源：
  ~/.hermes/douyin_state/*.json    （抖音）
  ~/.hermes/zhihu_states/*.json    （知乎）
  ~/.hermes/bilibili_states/*.json （B站）
❌ 错误数据源：
  events/*.md（只有日期和摘要，没有视频ID）
  thinkers/*.md（博主档案，不是断点数据）
```

### Step 3：三平台交替抓取
**🔄 核心规则：三平台交替请求**
```
错误：抖音12人全跑完→B站3人→知乎4人（一口气30+请求必被限）
正确：抖音付鹏→B站宋鸿兵→知乎环中星鉴→抖音蒋宇飞→B站付鹏→知乎古都闲云→...
```

**对每个博主：**
1. 调用对应平台抓取模块（[抓取/抖音抓取]或[抓取/B站抓取]或[抓取/知乎抓取]）
2. 返回新视频列表
3. **立刻往/tmp/monitor_status.md写一行记录**

**🚨 铁律：每处理完一个博主，必须立刻写状态清单！这样中断后读文件就知道哪到哪了。**

### Step 4：请求频率控制
```
有新视频要转录 → 转录时间（1-2分钟）天然限速，不用sleep
0新不用转录    → 必须sleep(12秒)再请求下一个
知乎纯文字      → sleep(5秒)保险起见
```

### Step 5：失败自动重试
```
第一次请求 → 返回空/403/JSON失败
  ↓ sleep(10秒)
第二次请求 → 成功→✅ / 失败→继续
  ↓ sleep(15秒)
第三次请求 → 成功→✅ / 失败→标记"失败-待重试"
```

**知乎特别注意：** API间歇性返回空（尤其连续请求时），**不是cookie过期**。sleep(5-8秒)后重试即可。

### Step 6：输出汇总
```
✅ 成功抓取：X条
❌ 失败待重试：Y条
⏭️  0新跳过：Z条
```

## 抓取结果保存规则
```
抖音/B站新视频 → ~/.hermes/work/videos/{博主}_{ID}.mp4
知乎新文章     → ~/.hermes/work/texts/{博主}_{ID}.json
转录完成       → ~/.hermes/work/transcripts/{博主}_{ID}.txt
```

## 标题筛选规则
- 持仓沾边（芯片/养殖/黄金/恒科/永赢/平安等）→ 进管道
- 宏观/中观沾边（利率/地缘/猪周期/AI等）→ 进管道
- 不沾边（鸡汤/社会/个人生活/纯卖课）→ 跳过

## 调试模式输出
**每个博主必须输出（缺一不可）：**
1. 断点ID + 断点时间 + 断点标题
2. API返回状态（成功/失败/限流）
3. API返回的视频列表（前5条：ID+时间+标题）
4. 比对过程（断点ID位置+新视频判定）
5. 最终结论：0新 / X新（列新视频ID+时间+标题）

**🚫 禁止：** 直接说"API失败"跳过 / 直接给结论不给原始数据 / 3个博主合并输出
