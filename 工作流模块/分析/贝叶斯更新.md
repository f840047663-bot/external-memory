# 贝叶斯更新

## 输入
| 参数 | 类型 | 说明 |
|:----|:----|:----|
| 转录文本路径 | 列表 | 来自转录模块的成功文本路径 |

## 输出（返回值）
| 字段 | 类型 | 说明 |
|:----|:----|:----|
| 状态 | ✅/❌ | 更新是否完成 |
| 影响持仓 | 列表 | 被影响的持仓及P值变化 |
| 事件归档 | 路径 | events文件保存路径 |
| 失败原因 | 字符串 | 如果❌，返回原因 |

## 执行步骤

### Step 1：读取转录文本
FOR 每个文本路径 IN 转录文本路径:
  - 读取内容
  - 提取核心观点

### Step 2：宏观前置审核
- 读`00-宏观传导框架.md`
- 判断事件是否重大
- 如果重大 → 问用户是否跑审核模块

### Step 3：贝叶斯计算
- 读对应`positions/{资产}.md`
- 计算P值变化
- 更新持仓文件

### Step 4：事件归档
- 写`events/{日期}.md`
- 更新`23-贝叶斯更新日志.md`

## 返回值示例
```
{
  "状态": "✅",
  "影响持仓": [
    {"资产": "芯片ETF", "P值变化": "51%→54%", "原因": "霍尔木兹利好"}
  ],
  "事件归档": "events/2026-05-31.md"
}
```

## 失败处理
| 故障 | 解决 |
|:----|:-----|
| 文本读取失败 | 跳过该文本，继续处理其他 |
| P值计算错误 | 返回❌，提示手动检查 |
| 文件写入失败 | 返回❌，检查权限 |
