# 日报汇报

## 输入
| 参数 | 类型 | 说明 |
|:----|:----|:----|
| 抓取结果 | 对象 | 来自Step ②的三平台结果 |
| 转录结果 | 对象 | 来自Step ③的转录结果 |
| 贝叶斯结果 | 对象 | 来自Step ④的更新结果 |

## 输出（返回值）
| 字段 | 类型 | 说明 |
|:----|:----|:----|
| 状态 | ✅/❌ | 日报是否发送成功 |
| 邮件路径 | 字符串 | 日报文件保存路径 |
| 失败原因 | 字符串 | 如果❌，返回原因 |

## 执行步骤

### Step 1：汇总数据
- 抓取统计：各平台新视频数
- 转录统计：成功/失败数
- 贝叶斯统计：影响持仓及P值变化

### Step 2：生成日报文件
```markdown
# 监控日报 {日期}

## 抓取结果
- 抖音：X个新视频
- B站：Y个新视频
- 知乎：Z篇新内容

## 转录结果
- 成功：N个
- 失败：M个

## 贝叶斯更新
- 芯片ETF：51%→54%
- ...

## 异常记录
- ...
```

### Step 3：发送邮件
```bash
# 用邮件发送器发送日报
python3 ~/.hermes/scripts/send_email.py \
  --to feng202210062126@qq.com \
  --subject "监控日报 {日期}" \
  --body {日报内容}
```

## 返回值示例
```
{
  "状态": "✅",
  "邮件路径": "~/桌面/每日视频总结/2026-05-31_日报.md",
  "发送时间": "2026-05-31 22:30:00"
}
```

## 失败处理
| 故障 | 解决 |
|:----|:-----|
| 邮件发送失败 | 返回❌，保存本地文件 |
| 汇总数据缺失 | 用"无数据"占位，不中断流程 |
