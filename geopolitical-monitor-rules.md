## 三、地缘事件↔持仓映射表

```
事件 → 逻辑链 → 影响持仓 → 操作建议
```

```

    ↓
黄金避险需求下降 → 金价跌
    ↓
恒生科技短期受情绪提振但中期仍受流动性压制
```

| 持仓 | 影响 | 建议 |
|:----|:-----|:-----|
| 芯片 020628 🔴 | 短期利空（情绪冷却→回调） | 等回调到位=入场机会 |
| 黄金 实物 | 利空（避险需求降） | 持有不动，实物不折腾 |
| 恒生科技 009854 | 短期情绪利好，中期不变 | 不操作，等右侧 |
| 养殖 008765 | 无直接影响 | 不理 |

```
关税加码 / 制裁升级
    ↓
国产替代预期升温 → 芯片ETF涨（但已过热）
    ↓
恒生科技被带崩（外资流出）
    ↓
黄金避险需求上升 → 金价涨
```

| 持仓 | 影响 | 建议 |
|:----|:-----|:-----|
| 芯片 020628 | 短期利好（但已经涨了，追高危险） | 不追，等回调 |
| 黄金 实物 | 利好 | 持有 |
| 恒生科技 009854 | 利空 | 更不买 |
| 养殖 008765 | 无直接影响 | 不理 |

### 场景3：美联储转鸽
```
降息/放水信号
    ↓
美元走弱 → 外资回流新兴市场
    ↓
```

| 持仓 | 影响 | 建议 |

## 四、输出格式（有信号时生成）

只在有触发信号时出分析，格式控制在300字以内：

```
━━ 地缘信号 YYYY-MM-DD ━━
🔴/🟡 信号等级

事件：一句话描述
逻辑链：2-3句
关联持仓：
  - 芯片020628：XXXX（影响+建议）
  - 黄金实物：XXXX
  - 恒生科技009854：XXXX
  - 养殖008765：无影响（跳过）

来源：闭眼看世界 / 其他博主
```

## 五、日报联动规则

- 有地缘信号且影响当日持仓时才在日报末尾加一句
- 没有或无关 → 不提，不占日报空间
- 信号对操作有直接影响时（如芯片买入窗口出现）→ 单独输出分析，日报里引用

## 六、监控接入方式

1. 用户给链接 → 加入 `douyin-monitor.py` 博主列表（标记 `intl` 类别）
