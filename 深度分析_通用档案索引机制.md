# 深度分析通用档案索引机制
# 任何深度分析完成后必须建立此索引

## 使用规则
1. 任何深度分析（博主/主题/事件）完成后，必须在 ~/.hermes/external_memory/ 建对应的 _档案索引.md
2. 索引必须记录：原始数据来源、分析报告路径、小龙审核记录、看板备份路径
3. 索引必须同步记录到L2记忆（hindsight_retain），格式："[分析对象]完整档案索引: [文件路径]"
4. 看板里涉及该分析的内容，必须标注"详见:xxx_档案索引.md"
5. 以后要看任何分析的来源，先查L2记忆→找到索引文件→索引文件里有所有原始路径
