# 每月审查清单

> 执行时机：每月1号自动检查，或用户说"审查"时执行
> 执行方式：逐项过，不合格的直接修

---

## 审查项

### 1. 断点文件完整性
- [ ] 断点文件有19人（跟thinkers/INDEX.md一致）
- [ ] 每人有"最后处理"日期
- [ ] 断点日期 ≤ Excel看板最后一条事件日期

### 2. P值准确性
- [ ] Sheet2 Row3每个持仓P值 = 50% + 事件链累加
- [ ] 有效P = P + 利率折价
- [ ] 虚P事件有【虚P】标签 + 橙色填充
- [ ] 没有硬编码的P值

### 3. 事件链完整性
- [ ] 每个事件有批注（Comment）
- [ ] 事件链从上到下时间递减（最新在上）
- [ ] 同天事件合并到一个单元格
- [ ] 没有丢失的事件（对比日志）

### 4. 叙事主线
- [ ] Row13-16有4条叙事主线
- [ ] 每条叙事有来源标签
- [ ] 每条叙事有完整便签（逻辑链+持仓影响+操作建议）

### 5. 待重仓监控
- [ ] 在总看板（表格1）内
- [ ] 有表头行
- [ ] 每个待重仓有P值+盈亏比+决策信号

### 6. 基金API数据
- [ ] 能查到持仓基金的每日涨跌
- [ ] 能查到历史涨跌
- [ ] 能计算波动率
- [ ] 数据跟看板联动分析
