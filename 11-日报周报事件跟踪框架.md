## 二、每日日报模板

```
📅 5月20日 周二 | 当日 ±X.X% (¥±XXX) | 净值 ¥XX,XXX
🔥 驱动：芯片ETF +2.3%·台积电4月营收超预期15% | 养殖ETF -0.8%·猪价续探底4元
🧠 博主共识：3/5博主看多AI硬件，伊娃重申猪周期反转要到2026底
📌 异常：无框架解释不了的波动
⚡ 红盘7/12项·仓位集中度62%(T0)
```

**每行规则：**

| 行 | 内容 | 自动化程度 |
|:---|:-----|:----------|
| 1️⃣ | 日期 + 当日总盈亏%+金额 + 最新总净值 | ✅ 净值API自动算 |
| 2️⃣ | 前2-3个涨跌幅最大的持仓 + 归因事件（一句话） | ✅ 事件匹配+博主语料 |
| 3️⃣ | 博主共识摘要（从当日转录提炼，1句话） | ✅ 转录摘要 |
| 4️⃣ | 异常波动（大于4%但无事件匹配的，标注⚠️） | ✅ 规则引擎 |
| 5️⃣ | 账户情绪：红盘比例 + T0仓位占比 | ✅ 自动统计 |

**如果当天T0梯队无显著波动（<1.5%）：**
```
📅 5月20日 周二 | 当日 +0.3% (¥+228) | 净值 ¥77,848
🔥 T0整体平盘，无显著驱动事件
🧠 博主关注：付鹏谈原油"肠梗阻"逻辑不变，白银等待机会
📌 无异常
⚡ 继续等待催化剂，按兵不动
```

## 三、事件归因流程（可编程）

```
每日输入：
┌──────────────────┐
│ 所有持仓实时净值  │──→ 计算当日涨跌幅 → 筛选|波动|≥2%
└──────────────────┘
┌──────────────────┐
│ 当日博主转录摘要  │──→ 关键词匹配（基金代码/行业/主题）
└──────────────────┘
┌──────────────────┐
│ 当日新闻标题流    │──→ 情感打分（正/负/中性）
└──────────────────┘
        ↓
  归因引擎：
  ① 找出波动>2%的持仓
  ② 匹配新闻+博主语料（同义词/代码/简称）
  ③ 方向一致（涨→正面事件，跌→负面事件）→ 主归因
  ④ 无匹配 → 标"⚠️ 无明确事件"
  ⑤ 博主共识压缩 → 三句话以内
```

**一期实现（先跑通）：**
- 净值API ✅ 已测试通过
- 博主转录 ✅ 已有daily-briefing skill
- 归因逻辑：我先手工匹配（人肉引擎），慢慢自动化
