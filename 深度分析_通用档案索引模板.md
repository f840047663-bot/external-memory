# 深度分析档案索引 - 通用机制
# 任何深度分析完成后必须建立此索引

## 机制说明
1. 任何深度分析（博主/主题/事件）完成后，必须在 ~/.hermes/external_memory/ 建立对应的 _档案索引.md
2. 索引文件命名格式：[分析对象]_档案索引.md
3. 索引必须记录：原始数据来源、分析报告位置、小龙审核记录、看板备份位置、核心结论
4. L2记忆必须存一条指向索引文件路径的记录
5. 看板里涉及该分析的内容，必须标注"详见:xxx_档案索引.md"
