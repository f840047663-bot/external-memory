## 二、各平台详细流程

### B站（bilibili_monitor.py）
- API端点：`https://api.bilibili.com/x/space/wbi/arc/search`
- 需要WBI签名（mix_encryption密钥）
- 分页：pn=1,2,3...

### 抖音（douyin-monitor.py）
- 使用yt-dlp下载音频
- 需要Netscape格式cookie：`/tmp/douyin_netscape_cookies.txt`
- 状态文件：`~/.hermes/douyin_states/`

### 知乎（zhihu_monitor.py）
- API端点：
  - 用户信息：`/api/v4/members/{token}`
  - 文章列表：`/api/v4/members/{token}/articles`
  - 回答列表：`/api/v4/members/{token}/answers`
- 需要cookie header
- 不用搜索，直接拼URL

## 三、Cookie提取（统一流程）

### 启动Chrome调试模式（⭐ 2026-05-30 最终版）
```bash
# 用local-chrome-cdp-bridge skill
# 或手动启动：
google-chrome --remote-debugging-port=9222 --user-data-dir=/tmp/chrome-debug
```

### 提取cookie
```bash
# 1. 先确保Chrome调试模式运行中
# 2. 打开目标网站并登录
# 3. 用CDP提取cookie
python3 -c "
import urllib.request, json
req = urllib.request.Request('http://127.0.0.1:9222/json')
tabs = json.loads(urllib.request.urlopen(req).read())
# 找到目标标签页，提取cookie
"
```
