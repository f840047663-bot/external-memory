- 一季度结婚比2013年同期↓60%

- **社消零售4月转负**（此前传+0.2%，南半球数据更差）
- **居民举债创历史新低** → 没人借钱了
- **农民工收入增速历史新低** → 最底层基本盘在崩

**核心逻辑链（「抗税」闭环）：**
```
消费弱 → 企业赚不到钱 → 税基萎缩
    ↓
政府加大征管力度
    ↓
中小企更难受 → 裁员/降薪
    ↓
就业更差 → 消费更弱 → 经济进一步下沉
```

**关键增量（vs 之前已知数据）：**
