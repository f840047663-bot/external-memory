# Rick Rule访谈：铜油铀长期看多 + 黄金长短分化 + AI对资源行业的免费期权

**信源：** 微信转发 | Rick Rule访谈（逆向资源投资人）
**时间：** 2026-06-25
**核心判断：** 铜/油/铀供给端存在长期刚性瓶颈，当前资源股估值未计入远期涨价+AI降本，属于隐含免费期权。黄金短空长多。铀是AI算力的意外赢家（最看好的细分）。

## 核心逻辑拆解（框架级判断，非短期交易信号）

### 一、铜 & 原油：供给端长期刚性瓶颈

**核心逻辑链：** 矿业长期勘探投资不足 → 铜矿从勘探到投产16-17年 → 未来10年新增供给很难释放 → 仅不含AI需求的铜需求就超过人类历史累计开采量 → AI数据中心带来额外增量 → 供给缺口结构性格局。

**关键定量：**
- 铜矿勘探到投产周期：16-17年
- 2026-2050全球所需铜量 > 人类历史累计开采量
- 资源企业估值用8%折现率NPV模型 → 11-12年后的远期现金流几乎不计入股价
- 远期资源涨价 + AI降本增效 + 勘探潜力：当前全部定价为0（免费看涨期权）
- 美元长期购买力贬值（类比70年代）→ 名义大宗商品中长期上行压力

**对油气行业的补充：** 油气行业长期资本投入缺口巨大，地缘冲突进一步压制产能修复。即便油价阶段性回落到65美元，供给结构性短缺并未消除。

### 二、铀：AI算力的"意外赢家"（Rick最看好）

**推理链：** AI数据中心需要24/7稳定基荷电力+低碳 → 风光间歇性无法满足 → 核能是唯一同时匹配的能源 → 铀需求确定性增长。

**三个支撑逻辑：**
1. **地缘能源安全：** 铀能量密度极高，小国少量储备可满足数年工业用电；油气/风光无法长期储存
2. **SMR商业化落地：** 比尔盖茨怀俄明项目推进，新一代堆可循环利用核废料
3. **时间维度：** 短期（2026）行情未必立刻兑现，但未来5-10年需求确定性极高

### 三、黄金：无法确认底部，长短逻辑分化

| 维度 | 方向 | 逻辑 |
|:----|:----|:------|
| 短期 | 🔴偏空 | 美国高名义利率支撑美元；流动性挤兑时黄金因流动性好常被优先抛售（无差别杀跌）→ 不能确认当前是底部 |
| 长期 | 🟢利多 | 美国巨额债务利息压力→最终倒逼宽松→牺牲美元购买力→黄金系统性重估 |

**Rick当前操作：**
- 实物黄金：持续分批储蓄，对短期价格不敏感
- 黄金股票：不再加仓大型矿企，新资金投向高风险初级勘探矿企（Junior）——近期大量优质钻探结果带来赔率

### 四、AI对资源行业的影响

**两点：**
1. AI可高效处理地球化学/构造/钻井数据，提升勘探效率，但无法完全替代野外实地勘探
2. 当前矿业股估值完全没有计入AI带来的长期降本增效红利——属于隐含免费期权，但兑现周期以十年计

### 五、客观风险（原文未充分提及）

1. 全球深度衰退 → 直接压制铜/原油/铀需求，拉长基本面兑现时间
2. 美联储降息节奏超预期/美元走强持续时间超预期
3. 核能政策黑天鹅（安全事故/监管收紧）→ 直接打压铀板块
4. Junior初级矿企风险极高：钻探数据好看≠经济开采价值；融资/审批/商品价格波动均可导致归零
5. Rick Rule属于逆向资源投资人，观点存在明显周期多头倾向
