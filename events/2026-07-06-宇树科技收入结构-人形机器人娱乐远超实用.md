# 宇树科技收入结构拆解——人形机器人娱乐属性远超实用价值

**信源：** 未指明具体博主（用户发来的视频文字摘要，推断来自某财经分析视频）
**时间：** 2026-07-06
**核心判断：** 人形机器人当前收入结构中，娱乐/展示/教育占比>90%，真正用于制造/巡检等实用场景的收入仅2.6%，反映行业共性问题——机器人距离大规模商业落地仍有巨大差距。

## 核心观点

1. **收入结构明细（宇树科技2025年1-9月，总营收~6亿）：**
   - 科研教育（高校实验室二次开发）：73.6% → ~4.4亿
   - 商业展示/表演：17.39% → ~1.04亿
   - 行业应用：仅9.01% → ~5400万
   - **经上交所追问，剔除企业导览等娱乐性质后，真正用于制造/巡检等实用场景的收入仅2.6% → ~1600万**

2. **行业共性问题：**
   - 当前人形机器人"娱乐属性远超实用价值"
   - 机器人卖出去后大量用在表演、接待、展示，而不是进工厂干活
   - 距离大规模落地（工厂产线、物流仓储、巡检检修）仍有巨大距离

3. **与前述事件链的关系：**
   - 2026-07-04 陈立（宇树联合创始人）亚布力发言："机器人尚未大规模应用，主因是具身智能大模型未成熟" → 此数据提供了具体的量化证据
   - 2026-07-05 模型先生体验人形机器人："想买但觉得实用性不足" → 消费者端的印证
   - 2026-07-03 69家A股公司发风险提示针对人形机器人等热点 → 监管层面已意识到概念过热
